概要

体験型VRバッティングゲーム

実行環境

・Unity 2020 3.26f1
・SteamVR 2.73
・HMD：VIVE Pro Eye
・PERCEPTION NEURON 2.0

機能

・球種
ストレート、スライダー、フォーク、シュート、爆速
➝変化球は手元で落ちるように、オブジェクトを介して手元で変化。
・球速、反発係数、球数、球とバットのサイズを変更可能
・スコアを設定
・フィードバック（音と振動）を付加
